Allows a CMS users (eg shop admin) to create a URL (for WooCommerce only) with specific product(s) and quantity info. When clicked by a user this URL will load those products into the users cart and take them to the checkout page automatically. Works for simple, grouped and variable products.

== Description ==

Allows a CMS users (eg shop admin) to create a URL (for WooCommerce only) with specific product(s) and quantity info. When clicked by a user this URL will load those products into the users cart and take them to the checkout page automatically. Works for simple, grouped and variable products.
